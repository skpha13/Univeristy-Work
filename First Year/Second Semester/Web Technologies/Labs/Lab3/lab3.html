Laborator 3

CSS-afisare si asezare in pagina

Proprietatea visibility

visibility: hidden
elementul devine invizibil in pagina (dar i se pastreaza locul)
visibility: visible
elementul devine vizibil

Proprietatea display:

display: none
elementul este eliminat din pagină (a nu se confunda cu visibility: hidden;)
display: block
elementul ocupă intreaga latime disponibila; i se pot atribui proprietățile width și height
display: inline-block
nu ocupa intreaga linie, și accepta width, height, margin-top, margin-bottom
display: inline
elementul ocupa cat spatiu are nevoie, nu acceptă width si height sau margin-top și margin-bottom

Proprietatea position

position: static
valoarea implicita a elementului in document
position: relative
elementul poate fi repozitionat relativ fata de pozitia naturala (initiala) din pagina; i se pot atribui proprietățile top, right, bottom și left (se aplica in afara margin-ului);
celelalte elemente nu se vor re-poziționa pentru a face loc (apar suprapuneri); în poziția inițială a elementului rămâne un gol
position: fixed
elementul poate fi repozitionat relativ fata de viewport (ramane pe loc indiferent de scroll) i se pot atribui proprietățile top, right, bottom și left;
nu lasa un gol in pozitia initiala
position: absolute
elementul poate fi repozitionat relativ fata de ultimul parinte cu pozitie ne-statica; i se pot atribui proprietatile top, right, bottom si left;
dacă nu are niciun parinte ne-static, se poziționează după viewport
position: sticky
se face toggle intre fixed si relative la scroll pentru a pastra elementul in pagina; i se pot atribui proprietățile top, right, bottom și left

Proprietatea z-index

z-index: numar intreg
defineste nivelul pe care se afla elementul
functioneaza doar pe elementele pozitionate (position diferit de static)
poate avea valori pozitive sau negative

Proprietatea float

  • are efect pentru elemente de tip bloc, inclusiv imagini (pentru elemente inline nu functioneaza, cum ar fi elemente span, q, i, b etc.)
  • float: left
    elementul este pozitionat in partea stanga a containerului sau
    float: right
    elementul este pozitionat in partea dreapta a containerului sau
    float: none
    pozitia implicita a elementului
    float: inherit
    elementul mosteneste valoarea proprietatii float de la parinte
  • restul continutului se va rearanja pe langa elementul asupra caruia i s-a aplicat float;
    clear:left / right / both anuleaza efectul rearanjarii
  • elementele pozitionate absolut ignora proprietatea float
