LABORATORUL 11 | Tehnici Web

Tema 11.1

Se considera documentul html comanda.html.
Implementati o aplicatie server folosind Node.js si Express astfel incat:

  • la submiterea formularului cu post folosind "formidable" sa se salveze datele intr-un fisier produse.json sub forma unui vector de obiecte avand proprietatile " produs", "pret", "cantitate", "poza" iar datele de tip file sa se uploadeze in folderul "poze". Raspunsul de la server va fi un mesaj cu produsul adaugat.
  • la cereri get catre "comenzi" se vor prelua datele din fisierul "produse.json" si se va genera folosind EJS pagina produse.ejs.

Tema 11.2

Se considera documentul html carti.html.
La click pe butonul "Generare", se va genera dinamic un tabel cu 4 coloane, avand antetul "Titlu", "Autor", "Pret" si "Descriere" iar datele din corpul tabelului vor fi preluate printr-o cerere Ajax, din fisierul carti.json.

Tema 11.3

Documentul HTML mail.html contine doua elemente input de tip text avand etichetele "Email" si "Subiect", un element de tip textarea cu eticheta "Mesaj" si un buton cu textul "Trimite mail".
Implementati o aplicatie server folosind Node.js si Express astfel incat la submiterea formei cu metoda POST sa se trimita un mail la adresa introdusa in campul "Email" si avand ca subiect
textul introdus in campul "Subiect" iar continutul mesajului sa fie textul introdus in campul "Mesaj" (folositi modulul nodemailer).
