Laboratorul 2

CSS = Cascading Style Sheets

  • limbajul folosit pentru stilizarea unui document HTML
  • codul CSS este format din reguli CSS

Sintaxa unei reguli CSS:

selector{proprietate1:valoare1; proprietate2:valoare2;}

Codul CSS poate fi introdus:
<!-- Inline -->
<tag style="proprietate: valoare;">(in interiorul tagului de inceput)

<!-- In <head>, folosind tagul <style> -->
<head>
<style type="text/css">
selector { proprietate: valoare; }
</style>
</head>

<!-- In fisier extern -->
<head>
<link rel="stylesheet" type="text/css" href="style.css" />
</head>

Proprietati:

  • simple (o valoare)
  • reuniune de proprietati (valori separate prin spatiu ex: border: 3px solid red;)

Unitati de masura:

  • absolute : px, cm, mm, pt, in,..
  • relative : %(din dimensiunea parintelui), vh, vw, vmin, vmax (1% din dimensiunea ferestrei), em (fontul parinte), rem(fontul radacina)

BoxModel:

  • elementele pot fi inconjurate de un chenar (proprietatea border)
  • spatiul dintre element si chenar (proprietatea padding)
  • spatiul dintre chenar si elementele exterioare (proprietatea margin)
  • continutul elementului este definit prin dimensiunile width si height

Proprietati de baza si valori:

  • color: red sau #ff0000 sau rgb(0,255,255),..
  • text-align: center/left/right
  • vertical-align: top/bottom/middle/...
  • background:
    • background-color:culoare
    • background-image:url("URLImagine")
    • background-repeat: repeat/repeat-x/repeat-y/no-repeat
    • background-position: center/top/left/etc
    • background-size: dimensiune/cover/contain
  • border:
    • border-width:grosime
    • border-style:stil
    • border-color:culoare
    • border-radius:margini rotunjite
  • width: dimensiune(pixeli/ procent din parinte/vw )
  • height: dimensiune(auto pastreaza raportul inaltime-latime)
  • font:
    • font-familiy:familia de fonturi
    • font-size:dimensiunea
    • font-style:normal/italic/oblique/inherit
    • font-weight:normal/bold/int(100-900)
  • visibility: visible/hidden(invizibil cu pastrarea spatiului)
  • display:
    • block (latimea implicita a elementului egala cu a parintelui )
    • inline (mai multe pe linie, nu permite dimensionare si includere elem block)
    • inline-block (mai multe elemente pe linie, permite dimensionare si includere elem block)
    • none (invizibil -fara pastrarea spatiului)

Alte proprietati:

  • border-collapse: collapse/ separate (unifica sau nu chenarul)
  • list-style-type: none/circle/square/...

Selectori:

  • Simpli
    • * (toate elementele)
    • numeTag (tagurile cu numele numeTag (ex: p selecteaza toate paragrafele))
    • #numeId (tagul cu atributul id="nume")
    • .numeClasa (tagurile cu atributul class="numeClasa")
  • Combinati
    • element1 , element2 (elementul 1 si elementul2)
    • elementContainer elementContinut (elementulContinut din elementulContainer)
    • elementParinte > elementCopil (elementCopil direct al elementParinte)
    • element1 + element2 (elementul de tipul element2 este primul frate care urmeaza imediat dupa element1)
    • element1 ~ element2 (elementele de tipul element2 care sunt frati cu element1 si urmeaza dupa element1)
  • Pseudo-clase
    • selector:nth-of-type(2n+1) (al 2n+1 -lea element de tip selector)
    • selector:nth-child(2n)(selectorul care este al 2n-lea copil (fratii anteriori pot sa nu fie de tip selector))
    • selector:hover (elementul de tip selector pe care se afla mouse-ul)
  • Atribut selector
    • selector[atrib="val"](element de tip selector cu atributul atrib avand valoarea "val")
    • selector[atrib*="val"](element de tip selector cu atributul atrib avand valoarea ce contine cuvantul "val")
  • Pseudo-elemente
    • selector::before, selector::after (adauga continut inainte/dupa elementul selectat)
    • selector::first-line (prima linie a unui element block)
    • selector:first-letter (prima litera a unui text-element block)

Specificitate

  • se pleaca de la valoarea 0
  • se adauga
    • 1000 pt stilizare inline
    • 100 pt selectare prin id (#id)
    • 10 pt selectare prin clasa (.clasa), atribut ([atribut]), pseudo-clasa (:hover..)
    • 1 pt selectare prin element (p,..), pseudo-element(:before)
  • selectarea cu specificitate mai mare este prioritara, chiar daca este urmata de o alta selectare cu o prioritate mai mica
